The Monster Expert Advisor is built on the following technologies:

Nodejs:

The Monster Expert Advisor is built on Nodejs, which is the backend of the Expert Advisor and executes trades, analyzes the market, and handles user inputs such as "account numbers", "passwords" and "API keys".

Reactjs:

The Monster Expert Advisor uses Reactjs as its Frontend Interface and user inputs entered here (Website) are sent to the Nodejs Backend server for processing and analysis. The results are then displayed on the website.

Python:

The Monster Expert Advisor uses Python to fetch historical market data from different sources, combine that data, calculate the "probability" for that trade, recommend lot size, and generate signals.
